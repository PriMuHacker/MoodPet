# MoodPet 🐾

MoodPet is a mental health and self-care app that helps users build positive habits by taking care of a virtual pet.

Instead of overwhelming users with complex systems, MoodPet turns small daily actions into meaningful progress through emotional connection, gamification, and personalized interaction.

---

## 🌟 Features

### 🧠 Mood Check-Ins

* Users can log how they’re feeling using a variety of emotions
* Encourages self-awareness and reflection

### ✅ Daily Tasks

* Simple, realistic self-care tasks (e.g., drink water, go outside)
* Completing tasks rewards users with gems 💎

### 🐾 Virtual Pets

* Start with a pet and unlock more as you progress
* Each pet represents a new journey and fresh start

### 💬 Interactive Pet Companion

* Pets respond to user input and provide encouragement
* Suggest tasks based on how the user is feeling
* Creates a sense of companionship rather than just tracking habits

### 🔄 Per-Pet Progression

* Each pet has its own task experience
* Switching pets allows users to start fresh while maintaining past progress

---

## 💡 Inspiration

Mental health tools can sometimes feel overwhelming or impersonal. MoodPet was designed to make self-care feel approachable, engaging, and even comforting.

By creating a bond between the user and their pet, the app encourages consistency through small, meaningful actions.
